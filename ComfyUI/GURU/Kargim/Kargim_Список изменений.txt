1.0 и ранние версии:
* Добавил переключатель для VAE, чтобы можно было использовать вшитый в модель.
* Заменил ноды позитивного промта на новые с возможностью перевода.
* Добавил ноду подгонки размера для IP Adapter (влияет на результат).
* У Face Detailer и Upscale перенес выбор LORA внутрь модуля. Имхо, так удобнее и меньше вероятность забыть включить  выключить.
* Добавил ноду для Outpaint-а.
* В Upscaler добавил возможность прямой обработки с Face Detailer (чтобы запускать циклом основную цепочку разом).
* Перекрасил группы для лучшего восприятия. Разные правки внешнего вида.
* Перестроил системы включения/выключения Kohya, ControlNet и IPAdapter для генерации TXT2IMG, Inpaint, Face Detailer и Upscaler!

1.1
* Добавлен функционал Стилей как в Автоматике/Фордже.
* Переключатели Kohya, ControlNet и IPAdapter возвращены к старому виду (у нового вылез неприятный косяк, который я не смог исправить).
* Исправил ошибку в подключении стилей. У меня Clip брался с модели, а надо из Лоры.
* Некоторые правки интерфейса.

1.2
* Добавил к IPAdapter ноду для ручного выделения нужной области маской. Сделал включатель.
* Перевел группу TXT2IMG, Face Detailer, Inpaint и Upscale на новый более удобный подход к включению/выключению ControlNet и подобных узлов.
* Некоторые изменения во внешнем виде.

1.3
* Добавил в inpaint возможность превращения замаскированной области в шум. Включается/выключается. Может пригодиться в некоторых сценариях, когда нужно полностью перерисовать, но inpaint модель слишком сильно цепляется за исходное изображение.

1.4
* Исправил ошибку в Upscale. В CNet подавалась картинка напрямую из загрузчика, а надо было с ноды переключателя входного изображения.
* Заменил ноду для негативного промта (с выходом текста) в блоках Face Detailer, Inpaint, Upscale.
* Теперь в SD Prompt Saver (генерация и Face Detailer) промт будет выводиться со стилем (раньше стиль не учитывался).
* Небольшие оптимизации во всех нодах SD Prompt Saver (поубирал лишние ноды).
* Пришлось снова вернуть переключатели CNet и IPAdapter к старому виду. Новые не работают, если ноды Load Image пустые.

1.5
* Вынес Стили в отдельный модуль. Теперь они доступны для всех модулей сразу.
* Удалил группу Сравнение изображений. Я ей редко пользуюсь, а при необходимости собрать очень просто и быстро.
* Убрал в Face Det и Upscale отдельные ноды для Lora. Пусть остается только одна глобальная.
* Еще сократил количество нод в SD Prompt Saver (убрал дубликаты и перенес название ckpt и vae в группу загрузчиков).
* Добавил для CNet в Inpaint возможность загрузить внешнее изображение.
* Переделал в INPAINT блок генератора шума. Применил другую ноду "Fill Masked Area", упростил структуру. Результат должен быть лучше.

1.6
* Добавил в INPAINT правку рук через Hand Refiner. Не всегда отрабатывает отлично, но в разы лучше, чем без него мучиться.
* Добавил в Hand Refiner возможность переключения на ручную маску (иногда полезно).
* Добавил дополнительный ручной переключатель промтов в стилях (на случай если автоматика будет срабатывать некорректно).
* Заменил ноды позитивного промта на вариант без conditioning (для удобства).
* Добавил BUS для загрузчиков и перекинул все подключения. Так линий на схеме стало поменьше.
* Добавил ноды для вывода итоговых промтов в группе стилей. По умолчанию свернуты. Может пригодиться для проверки.

1.7
* Добавил в INPAINT переключение в режим обработки полного изображения (а не кропа по маске). Может пригодиться в некоторых сценариях.

1.8
* В генерацию изображения внедрил сразу два подстановщика лиц FaceID (XL и 1.5) и PulID (XL). Единовременно можно использовать только один из них. Требует установленной и рабочей библиотеки insightface!
* Также добавил оба подстановщика лица в группу INPAINT.
* Добавил подстановщикам лица ноду Prep Image For ClipVision. Не знаю, на сколько она тут действительно нужна, но пускай будет.

1.9
* Приступил к интеграции Flux. В группу загрузчиков добавил возможность использования Flux GGUF моделей. Ну и переключатель для них.
* Добавил ноду FluxGuidance. Вроде, она нужна только для F1.S моделей, но точно не знаю. Выключить просто.
* Поправил вывод названия модели в SD Prompt Saver (для Flux), но для GGUF пришлось выключить подсчет хэша (в настройках ноды).
* DualCLIPLoader оставил обычный. Потестил gguf версии t5xxl. Прибавки к скорости работы они не дают, проще юзать t5xxl_fp8_e4m3fn.
* Добавил загрузчик и переключатель для FP8 моделей. У меня он загружается дольше, но по времени работы сопоставимо.
* Куча разных мелких правок, которые пришлось вносить для Flux.
* Исправил кучу Switch с ошибками.
* Добавил ноду TeaCache для ускорения работы Flux (но у меня не работает).

1.10
* Переделал полностью механизм выбора моделей. Разделил XL и Flux на отдельные группы с простым переключением и понятным выводом.
* Переделал механизм CSV стилей. Теперь группу можно полностью отключить и все будет работать (если файла стилей нет)! Упростил вывод.
* Пересобрал все группы.
* Пересобрал с нуля все BUSы. Попробовал от другого нод пака, но пока что MaraScott лучше работают.
* Flux Guidance перенес в группу Flux. Вроде, подключил правильно, но есть нюансы.
* Убрал список изменений из схемы. Буду вести отдельно.

1.11
* Заменил DualCLIPLoader на его GGUF версию. Оказывается, он универсальный и работает с обоими форматами (так удобнее).
* Небольшой редизайн группы загрузчиков Flux.

1.12
* Заменил ноду для ускорения Flux с TeaCache на Apply First Block Cache (WaveSpeed). Она более универсальная и работает с SD XL (в схему не добавлял, только потестировал).
* Небольшой редизайн группу загручиков Flux.

1.13
* Заменил ноды "Prep Image For ClipVision" для IPAdapter на "Enchance And Resize Hint Images" (в 2 местах). Оказалось, что старые автоматически резали разрешение до 224*224. Это же ппц просто.
* Удалил ноды "Prep Image For ClipVision" для лицозаменителей по той же причине.
* Изменил настройки у нод "Image Scale Down To Size" для лицозаменителей. Теперь изображение будет подстраиваться под 512 пикселей по наименьшей стороне (всегда легко изменить).
* Добавил еще один лицозаменитель InstantID в группы генерации и Inpaint. Работает на базе SD XL.
* Добавил ноду для ускорения WaveSpeed и для XL.
* Разные мелкие правки.
